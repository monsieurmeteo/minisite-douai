# ⚡ Automatisation & Archivage avec Supabase (Guide Complet)

Ce guide permet de rendre vote site 100% autonome et de créer votre propre base de données météo.

## 1. Créer un projet Supabase (Gratuit)
1. Allez sur [supabase.com](https://supabase.com) et créez un compte.
2. Créez un projet "minisite-meteo".
3. Notez l'URL du projet et la clé `service_role` (dans Project Settings > API).

## 2. Préparer la Base de Données
Ouvrez l'éditeur SQL de Supabase et lancez ce script pour créer les tables :

```sql
-- 1. Table pour le Token (Secret)
CREATE TABLE secrets (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  name TEXT UNIQUE NOT NULL,
  value TEXT NOT NULL,
  updated_at TIMESTAMP WITH TIME ZONE DEFAULT timezone('utc'::text, now())
);
INSERT INTO secrets (name, value) VALUES ('meteo_token', 'empty');

-- 2. Table pour l'Historique Météo
CREATE TABLE observations (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  station_id TEXT NOT NULL,
  station_nom TEXT,
  t FLOAT, -- Température
  u INT,   -- Humidité
  ff FLOAT, -- Vent
  rr FLOAT, -- Pluie
  date_obs TIMESTAMP WITH TIME ZONE NOT NULL,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT now()
);

-- Index pour accélérer les recherches par station et date
CREATE INDEX idx_obs_station_date ON observations(station_id, date_obs);
```

## 3. Créer le Script Automatique (Edge Function)
Ce script va tourner toutes les heures pour récupérer les données et les stocker.

Code (TypeScript pour Deno) :
```typescript
import { serve } from "https://deno.land/std@0.168.0/http/server.ts"
import { createClient } from 'https://esm.sh/@supabase/supabase-js@2'

serve(async (req) => {
  // CONFIGURATION
  const SUPABASE_URL = Deno.env.get('SUPABASE_URL')
  const SUPABASE_KEY = Deno.env.get('SUPABASE_SERVICE_ROLE_KEY')
  const MF_CLIENT_ID = Deno.env.get('MF_CLIENT_ID')
  const MF_SECRET = Deno.env.get('MF_SECRET')

  const supabase = createClient(SUPABASE_URL, SUPABASE_KEY)

  try {
    // 1. OBTENIR TOKEN
    const tokenResp = await fetch('https://portail-api.meteofrance.fr/oauth2/token', {
        method: 'POST',
        headers: { 
            'Authentication': 'Basic ' + btoa(MF_CLIENT_ID + ':' + MF_SECRET),
            'Content-Type': 'application/x-www-form-urlencoded' 
        },
        body: 'grant_type=client_credentials'
    })
    const tokenData = await tokenResp.json()
    const token = tokenData.access_token

    // Sauvegarder token pour le site web
    await supabase.from('secrets').update({ value: token }).eq('name', 'meteo_token')

    // 2. RÉCUPÉRER DONNÉES MÉTÉO (Paquet Obs)
    // On prend les données "infrahoraire" (toutes les 6 min)
    const obsResp = await fetch('https://public-api.meteofrance.fr/public/DPPaquetObs/v1/paquet/stations/infrahoraire-6m?format=json', {
        headers: { 'Authorization': 'Bearer ' + token }
    })
    const stations = await obsResp.json()

    // 3. FILTRER ET SAUVEGARDER (Pour économiser la DB gratuite)
    // Exemple: On ne garde que les stations du département 59 (Nord)
    const stationsNord = stations.filter(s => s.geo_id_insee && s.geo_id_insee.startsWith('59'))
    
    const rowsToInsert = stationsNord.map(s => ({
        station_id: s.geo_id_insee || s.id,
        t: s.t ? s.t - 273.15 : null,
        u: s.u,
        ff: s.ff,
        rr: s.rr_per,
        date_obs: new Date().toISOString() // Idéalement utiliser le vrai timestamp de la mesure
    }))

    if (rowsToInsert.length > 0) {
        const { error } = await supabase.from('observations').insert(rowsToInsert)
        if (error) throw error
    }

    return new Response(JSON.stringify({ success: true, saved: rowsToInsert.length }), { headers: { 'Content-Type': 'application/json' } })

  } catch (err) {
    return new Response(JSON.stringify({ error: err.message }), { status: 500 })
  }
})
```

## 4. Planifier (Cron Job)
Dans Supabase > SQL Editor :
```sql
select cron.schedule(
  'meteo-job',
  '0 * * * *', -- Toutes les heures pile (minute 0)
  $$
  select
    net.http_post(
        url:='https://votre-projet.supabase.co/functions/v1/meteo-job',
        headers:='{"Content-Type": "application/json", "Authorization": "Bearer VOTRE_SERVICE_ROLE_KEY"}'::jsonb
    ) as request_id;
  $$
);
```
